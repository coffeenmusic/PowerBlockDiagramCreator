Power Block Calculator

This bookmarklet loads the Power Block Diagram Current Calculator plugin into draw.io.

Installation

Step 1: Make sure your browser's bookmarks/favorites bar is visible.
Chrome: Ctrl+Shift+B  |  Firefox: Ctrl+Shift+B  |  Edge: Ctrl+Shift+B
Step 2: Drag the blue button below to your bookmarks bar:
Step 3: Done! The bookmarklet is now saved in your bookmarks bar.

Usage

1. Open app.diagrams.net (draw.io)
2. Open or create your power block diagram
3. Click the ⚡ Power Block Calculator bookmark in your bookmarks bar
4. You'll see an alert confirming the plugin loaded. Currents will now auto-calculate as you edit the diagram.
Note: The bookmarklet needs to be clicked each time you open or reload draw.io. It adds a "Recalculate Currents" option under the Extras menu and auto-recalculates on every diagram change.

For Desktop App Users

If you use the draw.io desktop application, you can load the plugin permanently instead:

  1. Launch draw.io with: "C:\Program Files\draw.io\draw.io.exe" --enable-plugins
  2. Go to Extras > Plugins...
  3. Click Add and select power_block_calculator.js
  4. Click Apply and restart draw.io (with --enable-plugins)
